# SQL Business Analysis Projects

A collection of SQL queries and data analysis projects solving real business problems.
Built while working as a Product Operations Analyst at ZoomRx and studying for Data/Business Analyst roles.

## Projects

### 1. Funnel Analysis - Study Cost Optimisation
**Problem:** Identify drop-off points and cost inefficiencies in a B2B research study workflow  
**Tools:** SQL, Excel, Power BI  
**Approach:** Wrote complex SQL queries to analyse funnel stages, identified where costs inflated, modelled scenarios  
**Outcome:** Reduced study cost from $30K to $15K (30%+ reduction), contributing to $1.8M+ in revenue

### 2. Vendor Cost Governance Analysis
**Problem:** Detect cost inflation patterns across vendor spend data  
**Tools:** SQL, Advanced Excel  
**Approach:** Queried vendor transaction data, identified anomalies, designed approval workflow  
**Outcome:** Delivered measurable cost savings, workflow adopted as standard process
